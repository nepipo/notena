🚀 6. Wann auf MAX? — Model-Upgrade-Roadmap

⚡
Sofort-Maßnahme: Prompt Caching in route.ts — 2 Zeilen, −47% Coach-Kosten dauerhaft

System-Prompt + Tools werden bei jeder Coach-Anfrage neu bezahlt (~2.800 Token = €0,0077/Msg). Mit cache_control: {type: "ephemeral"} am Nachrichten-Array werden diese gecacht: Cache-Write (1. Msg einer Session) kostet 25%, alle weiteren lesen den Cache für 10%. Bei 4-Msg-Session: −47%. Das ist die höchste ROI-Maßnahme in diesem Dokument. Kein Qualitätsverlust, kein Risk.
